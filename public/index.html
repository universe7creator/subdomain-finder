Frequently Asked Questions

Is this tool legal to use?
Yes, subdomain enumeration is a legitimate security research technique. However, you should only scan domains you own or have explicit permission to test. Always follow responsible disclosure practices and comply with applicable laws.
How fast is the scanning?
Scan speed depends on the domain and wordlist size. Typical scans complete in 5-30 seconds using multi-threaded DNS queries. The tool automatically handles rate limiting and DNS timeouts gracefully.
Can I use my own wordlists?
Yes! The tool comes with built-in comprehensive wordlists, but you can also upload and use custom wordlists for targeted reconnaissance on specific types of domains.
What export formats are supported?
You can export results to TXT (plain list), JSON (structured data with metadata), or CSV (for spreadsheet analysis). All formats include discovered subdomains, IP addresses, and HTTP status codes.
Does it detect wildcard DNS?
Yes, the tool includes wildcard DNS detection to filter out false positives when domains have wildcard records configured. This ensures you only see actual existing subdomains.
